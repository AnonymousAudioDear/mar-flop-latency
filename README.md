# MAR FLOPS and Latency Calculation

### Installation

Download the code:
```
git clone 
cd mar
```

A suitable [conda](https://conda.io/) environment named `mar` can be created and activated with:

```
conda env create -f environment.yaml
conda activate mar
```

### Calculate FLOPS
```
python3 cal_flops.py
```

### Calculate Latency
```
python3 cal_latency.py
```
